1
Multi-Sample Generation 5 x LLM Forward Pass
Generate 5 independent responses (temp = 0.7, max 512 tokens). Record per-token logits and softmax probabilities for each sample.
answer[0..4]
logits[0..4]
probs[0..4]
token_ids[0..4]
2
Semantic Clustering Up to 10 x LLM Forward Pass
Pairwise LLM equivalence checking with greedy agglomerative clustering. Groups answers by meaning, not exact wording.
clusters
num_clusters
cluster_sizes
3
Energy Teacher Computation Arithmetic Only
Boltzmann energy per sample: Ei = -mean(logits). Aggregate by cluster, normalise to [0, 1]. Main cluster share = confidence.
energy_score_raw
4
Entropy Teacher Computation Arithmetic Only
Shannon entropy over cluster size distribution: H = -Σ(|c|/N) · ln(|c|/N). Uses cluster structure only, ignores logits.
entropy_score_raw
5
Correctness Evaluation No Model Call
Normalised substring match of main answer against TriviaQA reference aliases. Used for evaluation only, NOT for training.
correctness
6
Hidden State Extraction 1 x LLM Forward Pass
Separate forward pass on (prompt + answer) with output_hidden_states=True. Extract at TBG position (last prompt token) and SLT position (2nd-to-last generated token). Shape: (33 layers, 4096 dims) each.
emb_last_tok_before_gen
emb_tok_before_eos
7
Logit Feature Extraction No Model Call
9 scalar summary statistics from sample 0: mean/min/std of chosen-token logits, mean/min/std of top-1/top-2 margins, answer length, mean/min softmax probability.
logit_feats (9 features)
Output: One Training Record
Serialised as Python pickle  |  ~500 valid records from 1,000 questions  |  ~541 MB total dataset

Figure 7.2 — Per-question record generation pipeline. Each of the 1,000 TriviaQA questions passes through seven processing steps. Coloured step numbers indicate the computation type: blue for generation, orange for clustering, green for energy, purple for entropy, red for hidden state extraction. Orange tags on each step indicate whether an LLM forward pass is required. The final record contains all fields needed for probe training and evaluation.
