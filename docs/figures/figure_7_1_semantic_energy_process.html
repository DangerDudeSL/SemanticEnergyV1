Figure 7.1 — Semantic Energy process diagram. A user prompt generates five independent samples via stochastic decoding. Pairwise semantic equivalence checking groups the samples into meaning-based clusters. Two parallel teacher signals are then computed from the same clustering: the energy teacher (green) aggregates per-token logit magnitudes by cluster, weighting by token confidence; the entropy teacher (purple) computes Shannon entropy over cluster sizes, ignoring logit magnitudes. Solid arrows indicate primary data flow; dashed arrows indicate secondary input (both clusters feed both teachers).
